💍Wedding Slideshow - Manifest Generator

📋 How to Use

  1. Use the file inputs below to select images, videos, and songs from your folders
  2. The manifest.json will be generated automatically
  3. Copy the JSON from the output box
  4. Create a new file named manifest.json in your wedding_data/ folder
  5. Paste the JSON content and save
  6. Open index.html in your browser (no server needed!)
